.score File Layout

This is an example of how our .score files are structured, featuring different musical elements:

<?xml version="1.0" encoding="utf-8"?>
<score bpm="120" beatspermeasure="4">
    <instrument instrument="Drums">
        <!-- Measure 1 -->
        <note measure="1" beat="1" duration="0.5" type="bass"/>
        <note measure="1" beat="2" duration="0.5" type="snare"/>
        <note measure="1" beat="2" duration="0.5" gain="1.0" type="snare">
            <effect which="chorus" wet=".5" dry=".5" />
        </note>
        ...
    </instrument>
</score>

Components

Piano Component

Owned by: Muhammad

The piano component synthesizes melodic piano sounds with dynamic velocity control, simulating expressive playing. Each note's timing, pitch, and intensity are manipulated to create a realistic piano feel. This component is foundational to the musical piece, providing harmonic structure.

Supported Grading Elements:

  • Plays piano notes
  • Envelope generation
  • Basic dynamics
  • Advanced dynamics
Download Piano Score File

Organ Component

Owned by: CJ

The organ component generates sustained tones with modulated overtones, recreating a classic organ sound. By controlling parameters such as tonewheel harmonics and vibrato, this component adds warmth and depth, often used to enrich the harmonic texture within the composition.

Supported Grading Elements:

  • Tonewheels
  • Drawbars
  • Envelope generation
  • Vibrato

Drums Component

Owned by: Hamil

This component provides rhythm through synthesized drum sounds like bass, snare, and cymbals. With diverse timing, tempo, and intensity, it creates a rhythmic backbone that defines the tempo and pulse of the piece, incorporating both regular beats and complex rhythmic variations.

Supported Grading Elements:

  • Waveform playback from tables
  • Envelope generation
  • Varying pitch playback from tables
  • 20 distinct drum sounds
  • Synthesized drum sounds

Scratching Synthesizer Component

Owned by: Raama

The scratching synthesizer produces unique, percussive "scratching" effects often associated with turntablism. This component modulates pitch and speed dynamically to create rhythmic scratches, adding a modern and experimental element to the synthesizer ensemble.

Supported Grading Elements:

  • Audio playback on demand from the sequencer
  • Attack and decay with pitch variation
  • 2 Scratch methods simulated
  • 4 Scratch methods simulated

Effects Component

Owned by: Omar

The effects component processes audio with effects like noise gate, flanger, chorus, and compressor, enhancing spatial depth. This component receives streamed audio frames from the other mentioned components such as piano and/or drums, and applies effects. We utilized other parameters such as wet and dry to determine the output that our frame will receive. When determining the structure of our score file, we initially thought that applying effects on an entire score would be fine, but we later ended up processing effects on a note-by-note basis. Each note can have multiple effects, and we process them in parallel.

Supported Grading Elements:

  • Audio passes through component
  • 1 Effect
  • 3 Effects
  • Controllable effects send
  • 4 Effects (Noise Gate, Flanger, Chorus, Compressor)
